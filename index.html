Basic Tags

Tag Name Description
<html> HTML Creates a HTML Document
<head> Document Header Sets off the title & other info that isn't displayed
<body> Document Body Sets off the visible portion of the document
<title> Document Title Puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked

Formatting

Tag Name Description
<p> Paragraph Creates a new paragraph
<blockquote> Blockquote Puts content in a quote - indents text from both sides
<div> Divison Used to format inline content with CSS
<span> Span Used to format inline content with CSS

Text Tags

Tag Name Description
<h1/2/3/.../6> Headline Creates headlines -- H1=largest, H6=smallest
<em> Emphasize Emphasizes a word (usually processed in italics)
<strong> Strong Emphasizes a word (usually processed in bold)

Lists

Tag Name Description
<ul> Unordered List Creates an unordered list
<ol> Ordered List Creates an ordered list (start=xx, where xx is a counting number)
<li> List Item Encompasses each list item
<dl> Definition List Creates a definition list
<dt> Definition Term Precedes each defintion term/td>
<dd> Definition Data Precedes each defintion

Links

Tag Name Description
<a href="URL"> Anchor Link Creates a hyperlink to a Uniform Resource Locator
<a href="mailto:"> Anchor Link Creates a hyperlink to an email address
<a name="NAME"> Anchor Link Creates a target location within a document
<a href="#NAME"> Anchor Link Creates a link to that target location

Forms

Tag Name Description
<form> Form(ular) Defines a form
<select multiple> Scrolling Menu Creates a scrolling menu. Name=? sets the name of the menu, Size=? sets the number of menu items visible before user needs to scroll.
<select name=?> Pulldown Menu Creates a pulldown menu
<option> Option Sets off each menu item
<input type="checkbox"> Checkbox Creates a checkbox. Set name=? after the "checkbox" for the name and value=? defines the value associated with the input
<input type="submit" > Submit Button Creates a submit button. Value=? sets the text in the submit button
<input type="reset"> Reset Button Creates a reset button.

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Tag Name Description
<colspan> Column Span Defines how many columns a td element should span.
<rowspan> Row Span Defines how many rows a td element should span.
